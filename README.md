# Loan-Default-Analysis-PowerBI
Power BI project analyzing loan default risk using 250,000 records with Power Query data cleaning and DAX measures.

## Project Overview
This project analyzes loan default risk using Power BI. The dataset contains over 250,000 records and was cleaned and transformed using Power Query.

The dashboard helps understand factors affecting loan default such as income, employment type, credit score, and loan purpose.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
- GitHub

## Dataset
- 250,000+ records
- Financial and demographic data
- Loan details
- Credit score information

## Data Cleaning (Power Query)
The dataset was cleaned using Power Query with the following steps:

- Removed duplicate records
- Handled missing values
- Corrected data types
- Created calculated columns
- Transformed raw data into analysis-ready format

## Data Modeling
Relationships and calculated columns were created to improve analysis.

## DAX Measures
Several DAX measures were used including:

- Average Loan Amount
- Default Rate
- Total Loan Amount
- Income Analysis
- Risk Metrics

## Dashboards

### 1. Loan Default Overview
Shows loan distribution, default rate trends, and employment impact.

### 2. Applicant Demographics
Analyzes credit score, education level, and borrower profile.

### 3. Financial Risk Metrics
Shows year-over-year loan trends and financial risk indicators.

## Insights

- Higher credit score borrowers receive larger loans.
- Unemployed applicants show higher default rates.
- Loan demand varies year by year.
